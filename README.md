# Frontend Vue Test Task

Тестовое задание Саши Шахновой в студию Артемия Лебедева на позицию стажёра-фронтендера

### Инструкция по запуску

Клонировать репозиторий
```
git clone https://github.com/suppukerr/frontend-vue-test-task.git
```

Перейти в папук проекта

```
cd frontend-vue-test-task
```

Запустить в папке проекта 
```
npm run dev
```

Выполненное задание будет по ссылке

http://localhost:3000/
